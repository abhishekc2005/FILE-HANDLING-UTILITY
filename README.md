Internship Task 1 – File Handling Utility (Java)

About the Task

This project was completed as part of my Java development internship and focuses on implementing core file handling operations using the Java programming language. The objective of the task was to build a simple command-line-based utility that allows users to perform basic file operations such as writing content to a file, reading from a file, and modifying (appending to) the contents of an existing file. The program simulates a basic menu-based interface where the user is prompted to select the type of operation they want to perform and then provide the necessary input accordingly.

The task is centered around understanding how Java handles file I/O (Input/Output) operations using classes like FileWriter, FileReader, BufferedReader, and exception handling mechanisms. It helped me explore how to safely interact with files in a local environment, manage content in text format, and structure Java programs that are user-friendly and error-resilient.

Features Implemented

The following functionalities were implemented as part of the program:

1. Write to a File – Allows users to overwrite an existing file with new content or create a new file if it doesn't exist.

2. Read from a File – Reads and displays the file content line-by-line on the console.

3. Modify a File – Appends new content to the existing file without overwriting the old content.

4. Error Handling – Handles common exceptions such as FileNotFoundException and IOException gracefully, ensuring the program does not crash unexpectedly.

How It Works

After running the program, the user is prompted to:

1. Enter a file name (e.g., data.txt).

2. Choose an operation from a displayed menu:

   1 for Write

   2 for Read

   3 for Append

3. Follow the menu by entering the desired text when prompted.

4. The program then performs the action and confirms the success of the operation.

This simple structure makes the tool intuitive and beginner-friendly.
